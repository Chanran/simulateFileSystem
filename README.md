# simulate file system

## 环境依赖

- git: **2.9.3**
- nodejs: **7.2.0**
- npm: **3.10.9**

## 使用说明

`git clone git@github.com:Chanran/simulateFileSystem.git`

`cd simulateFileSystem`

`npm i -d`

`npm start`

## 使用的技术
- 页面:html,css,javascript,metro UI
- 交互:vuejs
- 后台:nodejs
- 套壳:electron
